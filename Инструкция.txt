"Height in meters" - Выставление начальной высоты полета объекта
"Throw speed in m/s" - Выставление начальной скорости полета объекта
"Angle" - Выставление начального угла полета объекта
"Throw at an angle" - При выставления данного флага появляется возможность указать угол полета и выбрать тип создания графиков
"Create graphs at a certain degree interval" -  При указании флага, при нажатии на кнопку "Create path" будут создаваться несколько графиков через указанный интервал градусов
"Period angle" - Выставление периода угла для создания графиков( 0 < period < 90)
"Сreate graph" - Создания графика(ов) при указанных настройка
Setting:
"Accuracy" - точность создания графиков( accyracy >= 100)
"Clear all graph" - очистить все графики
"Automatically clear charts when creating a new one(s)" - При указании флага, при создании нового графика будут очищаться предыдующие графики
